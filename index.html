Done as a project in the course TNCG13, SFX Tricks of the trade at Linköping University. OpenCV is used to find points in the face which then drive a character model in Maya.

Voxel Cone Tracing

Done as a project in the course TSBK03, Advanced Game Programming at Linköping University. Real time global illumination using scene voxelization and cone tracing.

Monte Carlo Raytracer

Done as a project in the course TNCG15, Advanced Global Illumination and Rendering at Linköping University. This is a ray tracer implemented in C++.

Blanda

Done under 24 hours as a hackathon entry in a team of four. My main task was shader programming and making sure the balls mix elegantly. This won first prize in the Liu Game Awards (http://liuga.se).

Subsurface Scattering

Done as a project in the course TNCG14, Advanced Computer Graphics, at Linköping University. A C++ OpenGL implementation of Subsurface Scattering

Carbon Dioxide Emissions Visualization

Done as a project in the course TNM048, Information Visualization, at Linköping University. Data on carbon dioxide emissions and electricity production is visualized using intuitive techniques.

Procedural Planet Generation

Done as a project in the course TNM084, Procedural Methods for Images, at Linköping University. User defined parameters generate a planet in real time using one of several procedural methods. This project won the award "Technical Excellence" at the C Awards competition (http://www.cawards.se).

QR-Code Detection

QuizQuest
